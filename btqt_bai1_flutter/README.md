# Gold Tracker

Ứng dụng theo dõi giá vàng & quy đổi theo thời gian thực.

## Thiết lập API key

Ứng dụng đọc API key từ file `.env` ở thư mục gốc dự án:

```
METALPRICE_API_KEY=your_api_key_here
```

## Tính năng chính

- Bảng giá vàng trực tuyến (mua vào/bán ra)
- Quy đổi vàng sang VND theo nhiều loại vàng/đơn vị
- Lưu lịch sử tra cứu
- Biểu đồ biến động 7 ngày
